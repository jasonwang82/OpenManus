## 📝 使用说明

### 快速开始

1. **启动服务器**
   ```bash
   cd /Users/jasonwang/workspace/OpenManus/output
   python3 -m http.server 8888
   ```

2. **访问应用**
   在浏览器中打开: http://localhost:8888/index.html

### 功能操作

1. **切换路线**
   - 点击顶部的"第一天"、"第二天"、"第三天"按钮
   - 查看不同日期的景点列表和地图

2. **查看景点详情**
   - 点击左侧景点列表中的项目
   - 或点击地图上的标记图标
   - 详情卡片会平滑滚动到页面底部

3. **关闭详情**
   - 点击详情卡片右上角的 × 按钮
   - 详情卡片将隐藏

4. **地图交互**
   - 拖动地图查看不同区域
   - 滚动鼠标滚轮缩放地图
   - 点击标记查看景点信息
