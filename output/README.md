# 北京三日旅游指南

一个精美的北京三日旅游网页，集成腾讯地图展示旅游景点位置和路线规划。

## 项目特性

- ✨ 精美的响应式设计，适配各种设备
- 🗺️ 集成腾讯地图JavaScript API
- 📍 三日旅游路线规划（皇城根下、长城雄风、皇家园林）
- 🎯 景点标记和路线可视化
- 📱 移动端友好界面
- 🏛️ 丰富的景点介绍和实用信息

## 三日旅游路线

### 第一天 - 皇城根下
1. 天安门广场 - 世界上最大的城市广场之一
2. 故宫博物院 - 明清皇家宫殿，世界文化遗产
3. 景山公园 - 俯瞰故宫全景的最佳位置

### 第二天 - 长城雄风
1. 八达岭长城 - 最具代表性的长城段落
2. 慕田峪长城 - 景色秀丽，游客相对较少
3. 明十三陵 - 明朝13位皇帝陵墓群

### 第三天 - 皇家园林
1. 颐和园 - 中国规模最大的皇家园林
2. 圆明园遗址公园 - 曾经的"万园之园"遗址
3. 天坛公园 - 明清皇帝祭天祈谷场所

## 快速开始

### 1. 配置腾讯地图API密钥

**重要步骤：** 需要申请腾讯地图API密钥才能正常使用地图功能。

#### 腾讯地图API密钥申请步骤：

1. 访问腾讯地图开放平台：https://lbs.qq.com/
2. 注册/登录腾讯云账号
3. 进入控制台，创建应用
4. 选择"JavaScript API"
5. 获取API密钥（Key）

#### 配置API密钥：

打开 `index.html` 文件，找到第7行：

```html
<script charset="utf-8" src="https://map.qq.com/api/gljs?v=1.exp&key=YOUR_API_KEY"></script>
```

将 `YOUR_API_KEY` 替换为你申请到的真实API密钥，例如：

```html
<script charset="utf-8" src="https://map.qq.com/api/gljs?v=1.exp&key=OB4BZ-D4W3U-B7VVO-4PJWW-6TKDJ-6PB5F"></script>
```

### 2. 本地运行

#### 方式一：直接打开文件
1. 直接用浏览器打开 `index.html` 文件
2. 即可查看网页效果

#### 方式二：使用本地服务器（推荐）
如果遇到地图加载问题，建议使用本地服务器运行：

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js 的 http-server
npx http-server -p 8000
```

然后在浏览器访问：`http://localhost:8000`

### 3. 部署到服务器

可以将整个项目上传到任何静态网站托管服务：
- GitHub Pages

- Netlify
- 或任何支持静态文件的服务器

## 功能说明

### 景点展示
- 左侧显示景点列表
- 右侧显示腾讯地图
- 点击景点可查看详细信息

### 地图交互
- 显示所有景点的标记点
- 用红色线条连接景点形成路线
- 点击标记可查看景点详情
- 自动调整地图视野以显示所有景点

### 详情信息
每个景点包含以下信息：
- 景点名称
- 详细介绍
- 开放时间
- 门票价格
- 建议游览时长

### 响应式设计
- 桌面端：左右分栏布局
- 移动端：上下布局，地图在上方
- 自适应各种屏幕尺寸

## 常见问题

### Q: 地图无法显示？
A: 请检查：
1. API密钥是否正确配置
2. 网络连接是否正常
3. 是否使用本地服务器运行（推荐）
4. 浏览器控制台是否有错误信息

### Q: 地图标记不显示？
A: 可能的原因：
1. API密钥权限不足
2. 网络问题导致地图资源加载失败
3. 检查浏览器控制台的错误信息

### Q: 如何修改路线信息？
A: 编辑 `js/main.js` 文件中的 `itineraryData` 对象，修改景点信息即可。

### Q: 如何自定义样式？
A: 编辑 `css/style.css` 文件，修改颜色、布局等样式。

## 自定义配置

### 修改主题颜色

在 `css/style.css` 中，主要使用以下颜色变量：
- 主色：`#c41e3a` (中国红)
- 深色：`#8b0000` (深红)
- 背景：`#f5f5f5` (浅灰)

### 添加新路线

在 `js/main.js` 中的 `itineraryData` 对象添加新的日期和景点：

```javascript
4: {
  title: "第四天 - 新路线",
  spots: [
    {
      name: "新景点名称",
      lat: 39.xxxx,
      lng: 116.xxxx,
      description: "景点介绍",
      openTime: "开放时间",
      price: "价格",
      duration: "游览时长",
      brief: "简短描述"
    }
  ]
}
```

### 修改地图样式

腾讯地图支持多种地图样式，可以在 `initMap()` 函数中修改 `mapTypeId`：
- `TMap.MapTypeId.ROADMAP` - 路网图
- `TMap.MapTypeId.SATELLITE` - 卫星图
- `TMap.MapTypeId.HYBRID` - 混合图
