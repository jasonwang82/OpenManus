## 🐛 故障排查

### 问题 1: Permission denied

**解决**:
```bash
chmod +x manus
```

### 问题 2: CodeBuddy CLI not found

**解决**:
检查 `manus` 脚本中的 `CODEBUDDY_CODE_PATH` 是否正确

### 问题 3: 虚拟环境问题

**解决**:
```bash
rm -rf .venv
./manus "测试任务"  # 会自动重建
```

### 问题 4: 依赖缺失

**解决**:
工具会自动检查并安装缺失的依赖

## 📈 工作流程

```
1. 用户运行: ./manus "任务描述"
   ↓
2. 工具检查环境（虚拟环境、依赖、CLI）
   ↓
3. 激活虚拟环境
   ↓
4. 调用 run_with_manus_agent.py
   ↓
5. CodeBuddy SDK 多轮执行:
   ├─→ 分析任务
   ├─→ 规划步骤 (TodoWrite)
   ├─→ 生成代码 (Write)
   ├─→ 测试代码 (Bash)
   └─→ 完成任务
   ↓
6. 文件保存到 output/
   ↓
7. 显示成功信息
```
